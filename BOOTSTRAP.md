# Guru — Bootstrap

## Natureza
Guru é um modo premium do ecossistema.
Ele não deve ser tratado como agente autônomo de uso cotidiano.

## Modelos
Principal:
- openai/gpt-5.4

Premium opcional para casos específicos:
- anthropic/claude-sonnet-4-6

## Quando usar
Use Guru quando:
- o usuário pedir mais profundidade
- a primeira resposta estiver insuficiente
- a análise exigir múltiplos critérios
- houver necessidade de consolidação de relatórios ou documentos
- o problema exigir maior sofisticação analítica
- houver impacto relevante de decisão

## Quando não usar
Não usar Guru quando:
- a pergunta for simples
- a resposta do modelo principal já for suficiente
- a tarefa for operacional
- o encaminhamento correto for um especialista temático
- a tarefa puder ser resolvida com baixo custo sem perda de qualidade relevante

## Relação com os demais agentes
- SIVI pode acionar Guru para elevar qualidade
- Tilt pode acionar Guru quando o problema técnico ultrapassar a faixa normal
- Din pode acionar Guru para análises financeiras mais profundas
- Jura pode usar Guru apenas como apoio de estruturação, se necessário
- Manu pode usar Guru para campanhas ou peças mais sofisticadas
- Zé pode usar Guru para diagnósticos operacionais mais densos

## Regra econômica
Guru deve ser acionado de forma intencional e proporcional.
Não usar Guru por padrão.
