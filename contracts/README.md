# contracts/

A pasta contracts/ é destinada apenas a contratos internos do sistema SIVI — artefatos e configurações que conectam SIVI com ferramentas e integrações (ex.: workflows n8n, templates para gotenberg, contratos machine-readable usados por automações).

Regras rápidas:
- Documentos analisados por pessoas (minutas, contratos revisados, arquivos de trabalho) devem ser salvos em docs/.
- Apenas arquivos de integração, templates ou recursos usados por ferramentas automáticas devem permanecer em contracts/.
- Se houver dúvida, salve em docs/ ou pergunte ao SIVI.
